# 🚀 Spring Boot User Order API

## 📌 Project Overview

This is a backend REST API built using Spring Boot that manages users and their orders.
It includes authentication, authorization, and clean architecture practices.

---

## 🛠️ Tech Stack

* Java 17
* Spring Boot
* Spring Security
* JWT Authentication
* Maven
* MySQL (or H2)

---

## ✨ Features

* User Registration & Login
* JWT-based Authentication
* Create, Read, Update, Delete Users
* Order Management System
* Exception Handling (Global)
* DTO Pattern & Layered Architecture

---

## 📂 Project Structure

src/
├── controller/
├── service/
├── repository/
├── entity/
├── dto/
├── security/
├── exception/

---

## 🔐 Authentication

This project uses JWT (JSON Web Token) for secure authentication.

---

## ▶️ How to Run

1. Clone the repo
2. Open in IntelliJ
3. Configure database in `application.properties`
4. Run the main class

---

## 📬 API Endpoints (Sample)

* POST /auth/register
* POST /auth/login
* GET /users
* POST /orders
